== README

- Start the app with *rails server* command
- on localhost, when you first login with an account it is automatically made admin
  so you can add new bike or edit or delete existing ones.
- when signing up for another accounts they dont have admin role.
- admin can add new bikes and should provide the picture url of the bike.
- the sidebar contains the search fields which can be searched with either of them
